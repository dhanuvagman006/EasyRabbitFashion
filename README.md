# EasyRabbitFashion Frontend

Modern React frontend for an online fashion store using Vite and Tailwind CSS.

## Features

- Home page with bold hero branding and featured products
- Shop page with product grid and category filter
- Product details page
- Cart page with quantity control, total calculation, and clear cart
- Cart persistence with localStorage
- Product data fetched locally from `/datasets/products.json`
- Fully responsive layout with smooth hover animations

## Dataset

Products are stored in:

- `public/datasets/products.json`

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router

## Run the project

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build:

```bash
npm run preview
```
